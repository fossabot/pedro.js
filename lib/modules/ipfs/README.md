# Pedro IPFS Package

This submodule is part of [pedro.js](https://github.com/jovijovi/pedro.js) project.

## Install

```shell
npm install @jovijovi/pedrojs-ipfs
```

or

```shell
yarn add @jovijovi/pedrojs-ipfs
```

## Usage

```typescript
import * as ipfs from '@jovijovi/pedrojs-ipfs'
```
