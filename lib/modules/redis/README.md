# Pedro Redis Package

This submodule is part of [pedro.js](https://github.com/jovijovi/pedro.js) project.

## Install

```shell
npm install @jovijovi/pedrojs-rdis
```

or

```shell
yarn add @jovijovi/pedrojs-rdis
```

## Usage

```typescript
import * as rdis from '@jovijovi/pedrojs-rdis'
```
