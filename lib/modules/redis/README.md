# Pedro Redis Package

This submodule is part of [pedro.js](https://github.com/jovijovi/pedro.js) project.

## Install

```shell
npm install @jovijovi/pedrojs-redis
```

or

```shell
yarn add @jovijovi/pedrojs-redis
```

## Usage

```typescript
import * as redis from '@jovijovi/pedrojs-redis'
```
