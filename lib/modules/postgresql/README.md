# Pedro Postgres Package

This submodule is part of [pedro.js](https://github.com/jovijovi/pedro.js) project.

## Install

```shell
npm install @jovijovi/pedrojs-pg
```

or

```shell
yarn add @jovijovi/pedrojs-pg
```

## Usage

```typescript
import * as pg from '@jovijovi/pedrojs-pg'
```
