# Pedro FSM Package

This submodule is part of [pedro.js](https://github.com/jovijovi/pedro.js) project.

## Install

```shell
npm install @jovijovi/pedrojs-fsm
```

or

```shell
yarn add @jovijovi/pedrojs-fsm
```

## Usage

```typescript
import * as fsm from '@jovijovi/pedrojs-fsm'
```
