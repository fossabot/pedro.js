# Pedro Types Package

This submodule is part of [pedro.js](https://github.com/jovijovi/pedro.js) project.

## Install

```shell
npm install @jovijovi/pedrojs-types
```

or

```shell
yarn add @jovijovi/pedrojs-types
```

## Usage

```typescript
import * as types from '@jovijovi/pedrojs-types'
```
