# Pedro Event Package

This submodule is part of [pedro.js](https://github.com/jovijovi/pedro.js) project.

## Install

```shell
npm install @jovijovi/pedrojs-event
```

or

```shell
yarn add @jovijovi/pedrojs-event
```

## Usage

```typescript
import * as event from '@jovijovi/pedrojs-event'
```
