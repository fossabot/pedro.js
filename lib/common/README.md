# Pedro Common Packages

This submodule is part of [pedro.js](https://github.com/jovijovi/pedro.js) project.

## Install

```shell
npm install @jovijovi/pedrojs-common
```

or

```shell
yarn add @jovijovi/pedrojs-common
```

## Usage

```typescript
import {
	auditor,
	cache,
	cert,
	config,
	context,
	log,
	metrics,
	options,
	security,
	sys,
	util,
	version
} from '@jovijovi/pedrojs-common'
```
